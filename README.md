## 備品管理システム

### 概要

このシステムでは、事務用品などの備品の在庫管理を行うことができます。
備品の在庫を登録や編集、削除を行うことができ、注文履歴を作成することができます。また、検索や絞り込みで目的のデータを素早く確認することができます。ユーザー管理も操作が簡単で、管理者機能でユーザー情報の編集制限をすることが可能です。


### 主な機能
* ログイン・ログアウト機能
* 通知画面(ホーム画面)
* 在庫一覧
* 備品新規登録、更新、削除、注文機能
* 備品検索機能、ソート機能
* 注文履歴画面
* 注文データ登録、編集、削除機能
* 注文データ検索機能、ソート機能
* ユーザー管理機能（アクセス制限付き）

### 開発環境
　PHP 8.0.
　MySQL 5.7
　Laravel 8.83
